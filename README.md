## Setup
1. Initialized node workspace
```
npm init
```

2. Install babel to have better JavaScript syntax
```
npm install --save-dev \
@babel/core \
@babel/cli \
@babel/preset-env \
@babel/preset-react
```

3. Install web pack dependencies
```
npm install --save-dev \
webpack \
webpack-cli \
webpack-dev-server \
style-loader css-loader \
babel-loader
```
